## GIF Özellikleri:
- Boyut: 64x64px veya 80x80px (icon boyutuna göre)
- Format: GIF
- Loop: Sonsuz döngü (önerilen)
- Optimizasyon: Küçük dosya boyutu için optimize edilmiş GIF'ler kullanın

## Kullanım:
GIF'ler hover durumunda otomatik olarak gösterilir. Eğer GIF dosyası bulunamazsa, statik icon gösterilmeye devam eder.
