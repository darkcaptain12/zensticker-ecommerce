# GIF Dosyaları Oluşturma Rehberi

Bu klasöre animasyonlu GIF dosyalarınızı ekleyin. Aşağıdaki kaynaklardan ücretsiz GIF'ler indirebilirsiniz:

## Ücretsiz GIF Kaynakları:

1. **Giphy** (https://giphy.com)
   - Arama: "truck icon", "shield icon", "award icon", vb.
   - Filtre: "Stickers" veya "Icons"
   - Boyut: 64x64px veya 80x80px

2. **Tenor** (https://tenor.com)
   - Icon animasyonları için arama yapın
   - Küçük boyutlu GIF'ler seçin

3. **LottieFiles** (https://lottiefiles.com)
   - SVG animasyonları (GIF'e dönüştürülebilir)

4. **Flaticon** (https://www.flaticon.com)
   - Animated icons bölümü

5. **IconScout** (https://iconscout.com)
   - Animated icons

## Gerekli GIF Dosyaları:

### Stats Section (80x80px):
- `truck.gif` - Mavi kamyon animasyonu
- `shield.gif` - Yeşil kalkan animasyonu  
- `award.gif` - Mor ödül animasyonu
- `clock.gif` - Turuncu saat animasyonu
- `users.gif` - Pembe kullanıcılar animasyonu
- `star.gif` - Sarı yıldız animasyonu

### Features Section (64x64px):
- `palette.gif` - Mor-pembe palet animasyonu

- `truck-feature.gif` - Turuncu-kırmızı kamyon animasyonu
- `headphones.gif` - İndigo-mor kulaklık animasyonu

### Hero Section (32x32px):
- `trending.gif` - Sarı trend animasyonu

- `sparkles.gif` - Mor kıvılcım animasyonu

### WhatsApp Button (64x64px):
- `whatsapp.gif` - Yeşil WhatsApp animasyonu

## GIF Özellikleri:
- **Format**: GIF
- **Loop**: Sonsuz döngü (infinite loop)
- **Boyut**: Icon boyutuna göre (32px, 64px, 80px)
- **Optimizasyon**: Küçük dosya boyutu için optimize edilmiş
- **Arka Plan**: Şeffaf (transparent) veya uyumlu renk

## Alternatif: CSS Animasyonları

Eğer GIF dosyaları bulunamazsa, sistem otomatik olarak statik icon'ları animasyonlu CSS ile gösterir (pulse animasyonu).

## Hızlı Test

GIF dosyalarını bu klasöre ekledikten sonra, sayfayı yenileyin ve icon'ların üzerine hover yapın. GIF'ler otomatik olarak gösterilecektir.
